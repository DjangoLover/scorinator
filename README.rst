Requirements
------------

  * Python 2.7+ (Not tested with prior versions of python)
  * Django 1.5+ (Not tested with prior versions of django)


Documentation
-------------

You can build the documentation locally or `view the documentation`_ on ReadTheDocs.

::

    pip install sphinx
    cd docs
    make html
